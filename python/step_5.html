Our first method

We are not going to add the first method with some basic logic. We start simple, with the ping endpoint. The desired behaviour is to receive a call from the pokerwars server, log it, and reply with a pong.

First we need to create a method linked to the ping endpoint. The syntax is def keyword followed by the method name and the parameter (none at the moment) in brackets and colon symbol.

Then inside the method we are going to log the call: we are using the printl command with the following message: 

'Received ping from pokerwars.io, responding with a pong'

And finally the interesting part, replying back with a pong JSON content. We are doing that setting the response.content_type to 'application/json' and invoking the return command with the JSON structure that we want to send back. E.g. 

return {"pong": True}

Now it's your turn, put together the 3 steps we described before, and you should be able to proceed to the next level.
