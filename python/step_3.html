Bottle

Bottle is a fast, simple and lightweight micro web-framework for Python. Now we are going to show you how to define endpoints, receive data from a API call and call to external APIs.

Let's focus of few interesting concepts in bottle:

  • Routing: Requests to function-call mapping with support for clean and dynamic URLs.
  • Templates: Fast and pythonic built-in template engine and support for mako, jinja2 and cheetah templates.
  • Utilities: Convenient access to form data, file uploads, cookies, headers and other HTTP-related metadata.
  • Server: Built-in HTTP development server and support for paste, fapws3, bjoern, gae, cherrypy or any other WSGI capable HTTP server.
