The starter projects

In our GitHub repository you can find starter projects for a number of languages. You are not limited to those, you can use any programming language you want as long as it's capable of sending and receiving HTTP requests.

For Javascript, our starter project is here. We will build that same starter bot now step by step. At the end, you will get an understanding of the game and the platform, and a working bot ready for you to implement your own unique strategy. How cool is that!
