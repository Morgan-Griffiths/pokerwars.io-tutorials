Welcome to pokerwars! In this tutorial you will learn how to build a simple bot. Let's get started!

The starter bots

In our GitHub page you can find starter bots for a number of languages. These are very simple versions of pre-built bots. You are not limited to those, you can use any programming language you want as long as it's capable of sending and receiving HTTP requests.

For Javascript, our starter project is here. In this tutorial we will build that same starter bot, step by step. At the end, you will get an understanding of the game and the pokerwars platform, and you'll have a working bot ready, which you can modify to implement your own unique strategy. How cool is that!
